#### 문제개요

다중 정렬하는 문제이다.

조건

1. 승률이 높을수록 먼저
2. 승률이 동일하면, 자신보다 무거운 사람을 이긴 횟수로 내림차순 정렬
3. 조건2가 동일하면, 자신의 무게로 내림차순 정렬
4. 조건3이 동일하면, 번호 오름차순 정렬

#### 접근방법

- 조건을 거는 것은 어렵지 않음.
- sort문 안에서 우선순위를 조건에 맞춰 걸면 됨

#### 실수한 것

- 문제를 제대로 읽지 않음
- "N" 비긴 것은 게임을 한 횟수에 포함되지 않음.
- 처음에 비긴 것 고려하지 않아 테스트케이스를 틀림
